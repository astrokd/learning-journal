# HTML/CSS book, Ch. 15, “Layout” (again; repeat of Class 4 reading)

> The liquid layout uses percentages to specify the width of each box so that the design will stretch to fit the size of the screen.

## Fixed layout CSS

``` CSS
body {
  width: 960px;
  margin: 0 auto;}
```


## Liquid layout CSS

```CSS
body {
  width: 90%;
  margin: 0 auto;
}
```
